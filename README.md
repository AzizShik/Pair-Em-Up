# Pair 'em Up

**A strategic number-matching puzzle game built with vanilla JavaScript**

Match identical numbers or pairs that sum to 10, strategically clear the grid, use powerful assists, and reach **100 points** before running out of moves!

Live Demo: https://azizshik.github.io/Pair-Em-Up/

## Features

- Three unique game modes: **Classic**, **Random**, and **Chaotic**
- Smart pair validation (same numbers or sum to 10)
- Bonus points for pairing fives (5 + 5 = +3 points!)
- Five powerful assist tools with limited uses
- Responsive design (380px – 1280px)
- Light / Dark theme toggle
- Full sound effects control
- Game state auto-save (continue after reload)
- High score table with last 5 games
- Timer, scoring, undo system, shuffle, hints, and more!

## Game Modes

- **Classic** – Numbers 1-9 placed in a beautiful sequential pattern
- **Random** – Same numbers, but completely shuffled
- **Chaotic** – 27 random digits from 1-9, duplicates allowed
